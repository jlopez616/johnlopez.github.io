Features

Defining the types, operators, and functions found in StoneScript

Types

Primitive Types

Number: COUNTERS
String: WORDERS
Boolean: YESNOS
Undefined: WHAT

Slightly Less Primitive Types

Object: THINGS
Array: STUFFS
Function: YABBADABBADOO

Variables

Let: ROCK
Const: BEDROCK

Operators

Arithmatic Operators

Addition (+): SQUISH
Subraction (-): RIP
Multiply (*): MANY
Divide (/): BREAK
Modulus (%): LEFT
Power (**): MANY MANY

Logical Operators

And (&&): AND
Or (||): OR
Not (!): NOT

Relational Operators

Equal (==): IS IS
Not Equal (!=): NOT IS
Less than (<): NOT SMASH
Greater than (>): SMASH
Less than or equal (<=): NOT SMASH OR IS
Greater than or equal (>=): SMASH OR IS

Statements

Loops

StoneScript features support for while and for loops.
While: WHILE
For: FOR

Conditionals

StoneScript features chainable if, else if, and else statements.
If: OOF
Else if: OOOF
Else : OOFF

Functions

Functions in StoneScript are lazily-evaluated, but will accept parameters and can be run over arrays. Arrays can be pattern matched and support currying.
Function: YABBADABBADOO
Return: GIVE
Arrays: STUFFS
[ / ]: CAVEIN / CAVEOUT
